# Industrial AI OS 面试演示指南

目标：在 5–8 分钟内展示一个完整、可操作、有工业业务逻辑的 AI 系统，而不是逐个浏览菜单。

## 演示前准备

1. 使用 Chrome、Edge 或 Safari 打开项目地址。
2. 建议先在首页点击“清空测试数据 / 重置演示环境”。
3. 使用演示账号登录：

```text
admin@personal-ai-os.local
123456
```

4. 如果没有配置真实 AI，保持 Mock 模式即可；界面会明确显示当前模式。

## 推荐演示路线

### 1. 登录与项目定位（约 30 秒）

- 展示演示账号登录。
- 说明系统是面向制造业与企业办公的 Industrial AI OS。
- 强调浏览器直接运行、响应式适配、Mock/API 双模式以及 AI Gateway 统一调用。

### 2. Dashboard 工业驾驶舱（约 45 秒）

- 展示今日订单、生产计划风险、设备台账、高负载设备、Agent 和系统状态。
- 点击“一键加载演示数据”，说明系统能快速恢复面试演示环境。
- 简要介绍数据会驱动生产计划和风险输出，而不只是静态数字卡片。

### 3. AI 生产计划助手（约 2 分钟）

1. 打开“生产计划助手”。
2. 点击“填充示例订单”，或下载 CSV 模板后导入订单。
3. 说明订单字段包括客户、产品、数量、交期、工艺、设备需求、优先级和状态。
4. 点击“AI分析订单”。
5. 展示生产计划、交期风险、设备负载、物料风险和每日安排。
6. 点击“生成生产日报”，再演示复制或导出 TXT。
7. 打开“设备台账”，说明排产会参考设备负载、适用工艺和维护状态。

推荐讲法：系统不是只把订单发给模型，而是先结构化订单和设备约束，再通过 AI Gateway 生成可人工确认的计划。

### 4. Excel 助手（约 1 分钟）

1. 打开“Excel助手”。
2. 加载验收示例或上传发货单。
3. 展示表格预览、产品明细识别、自动统计、查重和分类。
4. 强调系统只统计有效产品明细，不把电话号码、备注、合计说明等误当金额。
5. 演示 AI 业务分析或 Excel 导出。

### 5. PDF / OCR / PPT 助手（约 2 分钟）

PDF：

- 上传文本型 PDF，展示文件名、大小、页数、读取状态和文字预览。
- 点击“AI总结PDF”。
- 如无文字层，展示“该 PDF 可能是扫描件，请使用 OCR 图片识别”的明确提示。

OCR：

- 点击“加载示例”后点击“开始识别”。
- 展示进度、识别文字和发货单结构化字段。
- 演示导出 TXT、Word 或 Excel。

PPT：

- 输入主题“智能工厂数字化升级方案”、行业“机械制造业”、页数 8、用途“面试演示”。
- 点击“生成PPT大纲”。
- 展示每页标题、页面内容和建议视觉，并指出页面同时显示 AI Mode、Provider、Model。

### 6. 数据脱敏与 AI 历史（约 45 秒）

- 打开“数据脱敏”页面，粘贴一段包含手机号、邮箱、身份证号、客户名称、地址和金额的文本。
- 点击“一键脱敏”，再演示复制结果和导出 TXT。
- 打开“AI调用历史”或 AI 状态中心，展示最近调用、Mock / 真实 API、耗时、Token 和成本统计。
- 强调远程 AI 输入会有明确安全提示，适合企业演示而不是直接上传真实机密。

### 7. AI 设置中心（约 45 秒）

- 打开“设置”中的 AI Gateway。
- 展示 Mock、API、Hybrid、Local Model 模式。
- 说明真实 API Key 仅配置在 Vercel 环境变量，不写入前端。
- 展示 Base URL、Provider、Model、Temperature、Max Tokens 和超时配置。
- 不要在公开面试中展示真实密钥。

### 8. 系统验收中心（约 45 秒）

- 打开“系统验收中心”并点击“开始检测”。
- 展示 PDF 上传/总结、OCR 识别/导出、PPT Gateway/Mock、生产计划、CSV 和设备台账检查项。
- 说明验收中心用于把“按钮能点”升级为“核心链路有状态、有结果、可排查”。
- 补充说明：验收中心现在也会检查数据脱敏、AI 历史、AI 成本统计、Developer Mode 和安全文档。

## 收尾总结

推荐用一句话收尾：

> Industrial AI OS 将制造业订单、设备和文档数据接入统一工作台，通过 AI Gateway 提供生产计划与办公智能，并用 Mock、日志和验收机制保证项目在离线面试和真实 API 环境下都可演示、可扩展。

## 常见问题

- AI 没回复：检查 AI Mode、API 地址和 Vercel 环境变量；Mock 模式仍可完成演示。
- PDF 没文字：该文件可能是扫描件，转到 OCR 助手处理。
- OCR 首次较慢：浏览器首次加载识别资源需要时间；失败会显示 Mock OCR 结果。
- 页面数据太乱：回到首页重置演示环境后重新加载标准数据。
